        ## Validation Report: FB-003

### Summary
**PASS** — The `FlowOutputCell` data model, `render_flow_output_cell` renderer, `_output_section` helper, and `_child_returns_section` helper are all fully implemented with correct conditional rendering, 50-line truncation, XSS escaping, error styling, and child return pill formatting.

### Evidence

**Data Model (`flow_models.py:105-115`)**
- `FlowOutputCell` is `@dataclass(frozen=True)` with exactly 7 fields: `kind="output_cell"`, `stdout=""`, `stderr=""`, `child_returns=[]`, `has_errors=False`, `pane_id=""`, `invocation_id=""`
- `FlowChildCard` (`flow_models.py:83-102`) has all required pill fields: `depth`, `fanout_idx`, `status`, `error`, `input_tokens`, `output_tokens` (plus 10 more optional fields)

**Empty/Whitespace Guard (`flow_output_cell.py:16-18`)**
- `has_content = cell.stdout.strip() or cell.stderr.strip() or cell.child_returns` — whitespace-only strings evaluate to falsy after `.strip()`, and empty `child_returns` list is falsy → early return

**Conditional Section Rendering (`flow_output_cell.py:25-30`)**
- STDOUT rendered only when `cell.stdout.strip()` is truthy, with label `"STDOUT"` and `accent="var(--text-1)"`
- STDERR rendered only when `cell.stderr.strip()` is truthy, with `accent="var(--accent-child)"` and `error=True`
- CHILD RETURNS rendered only when `cell.child_returns` is non-empty

**50-Line Truncation (`flow_output_cell.py:52-55`)**
- `lines = text.strip().splitlines()` then `preview = "\n".join(lines[:50])` with overflow: `preview += f"\n... ({len(lines) - 50} more lines)"`

**XSS Prevention (`flow_output_cell.py:61`)**
- `f"{escape(preview)}"` — `escape()` from `html` stdlib applied to the preview before embedding in `<pre>` HTML

**Error Styling (`flow_output_cell.py:41-42`)**
- Error background: `"rgba(255,107,159,0.06)"`, error border: `"var(--accent-child)"`
- Non-error background: `"rgba(26,35,56,0.4)"`, non-error border: `"var(--border-1)"`

**Child Return Pills (`flow_output_cell.py:79-94`)**
- Color dispatch: `"var(--accent-child)" if child.error else "var(--accent-active)"` (line 80)
- `dN:fM` label: `f"d{child.depth}:f{child.fanout_idx}"` (line 87)
- Status badge: `"ERR" if child.error else "OK"` (line 90)
- Token count: `f"{child.input_tokens}in/{child.output_tokens}out"` (line 92)
- Pill shape: `border-radius: 999px` (line 84)

**Builder Integration (`flow_builder.py:184-206`)**
- `FlowOutputCell` constructed from `inv.repl_stdout`, `inv.repl_stderr`, and child summaries mapped to `FlowChildCard` instances
- `has_errors` derived from `bool(inv.repl_stderr.strip())`
